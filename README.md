## Я Python Backend Developer.
Python Backend-разработчик с осознанным подходом к созданию ПО. Стремлюсь писать чистый, поддерживаемый код и выстраивать правильную архитектуру даже в рамках pet-проектов.
* Инженерный подход: Пишу код с прицелом на поддержку и масштабирование. Уделяю внимание не только бизнес-логике, но и структуре БД, обработке ошибок, логированию и тестам. Понимаю важность разделения слоев (Clean Architecture) и работы с зависимостями.
* Стек и инструменты: FastAPI, PostgreSQL, SQLAlchemy 2.0, Alembic, Docker. Настраиваю CI/CD процессы на уровне репозитория: линтеры (ruff), black, pre-commit, Git Flow (feature-ветки, PR).
* Отношение к работе: Спокойно отношусь к код-ревью, воспринимаю его как точку роста. Быстро учусь, люблю доводить задачи до качественного результата.
* Цель: Ищу сильную команду, где смогу приносить пользу продукту и перенимать опыт у менторов и старших коллег.
